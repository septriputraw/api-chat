# api-chat
simple API 

# Requirements
## Node.js v14+
## PostgreSQL

# Running
run npm install

npm run migrate up

npm run start-dev 

# End-Points
## users
### POST /users
### GET /users/{id}
## authentications
### POST /authentications
### PUT /authentications
### DELETE /authentications
## chat
### POST /chat/send
### GET /chat
### GET /chat/{id}
